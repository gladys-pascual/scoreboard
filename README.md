I created a [**scoreboard**](https://score-me.netlify.app/) app using React.

## Key Features:

- Add and remove players
- Increase and decrease the scores of each player
- Highlight the winning player
- Display total players and total score
- Stopwatch

## Technologies used:

- JavaScript, React
- HTML
- CSS, SASS, Flexbox

## Libraries used:

- react
- node-sass
